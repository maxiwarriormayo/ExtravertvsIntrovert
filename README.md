# ExtravertvsIntrovert
Project to show the specific relationship between various situations and extraversion or introversion

# Tools Used
Pandas and Matplotlib

# description of what was done
I cleaned the data using pandas
Further analysis showed:

I also found out that the average friend circle size for introverts and extroverts using my dataset
